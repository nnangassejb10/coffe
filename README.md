📌 Présentation du projet

Ce projet est un site web moderne et interactif développé pour Velora Coffee, avec une approche orientée expérience utilisateur immersive, incluant des éléments 3D et une interface fluide.

L’objectif principal est de proposer une vitrine digitale attrayante mettant en valeur les produits et l’identité de la marque.

⚙️ Technologies utilisées

Le projet repose sur une stack moderne front-end :

React + TypeScript : pour une structure robuste et maintenable
Vite : pour un environnement de développement rapide et optimisé
Tailwind CSS : pour le design responsive et stylisé
shadcn/ui : pour des composants UI modernes et réutilisables
PostCSS : pour le traitement du CSS
Playwright / Vitest : pour les tests

🎨 Fonctionnalités principales
Interface utilisateur moderne et responsive
Intégration d’éléments 3D pour une meilleure immersion
Ajout d’images et de styles personnalisés
Intégration d’un favicon
Organisation modulaire du code avec composants réutilisables

📁 Structure du projet
source/ : contient les composants, styles et logique principale
index.html : point d’entrée de l’application
package.json : gestion des dépendances
tailwind.config.ts : configuration du design
vite.config.ts : configuration du build

🚀 Installation et lancement
npm install
npm run dev
